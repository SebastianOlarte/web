Estructura mínima de un programa en C

  1. Historia.
  2. Palabras claves.
  3. Comentarios.

Historia

Dennis Ritchie (creador del C - 1971) - Programación estructurada. El lenguaje C es versátil, conciso, y de nivel relativamente bajo. Es adecuado para la mayor parte de programación de sistemas y se ejecuta en muchas plataformas.
Uno de los objetivos originales de C era sustituir en gran medida el lenguaje ensamblador para las tareas más exigentes de programación de sistemas.

Palabras claves

Son los nombres que emplea el lenguaje para definir su estructura (ciclos, condiciones, etc.). Las palabras claves no pueden ser empleadas para definir variables y nombres de funciones, son de uso del lenguaje.
En C las palabras claves se especifican en minúscula.

Es importante tener en cuenta que los lenguajes C y C++ son sensibles a mayúsculas y minúsculas, es decir que si especificamos una palabra clave con alguna letra en mayúscula el compilador nos indicará que hay un error sintáctico.

Comentarios

Para comentarios de línea, se utiliza la doble barra //
// int Valor;
Un comentario de bloque es un conjunto de líneas no ejecutables. Se encierra entre /* al comienzo y */ al final.
